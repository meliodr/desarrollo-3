# Desarrollo 3 - Aplicacion Web con IA

Aplicacion web academica para registrar casos mediante un servicio web externo e integrar analisis de imagen con Amazon Rekognition.

## Funciones

- Formulario web para captura de datos.
- Carga de fotografia.
- Moderacion de imagen para bloquear contenido no apto.
- Reconocimiento de celebridades.
- Envio de datos al servicio web configurado.
- Evidencias simuladas para documentar el flujo
