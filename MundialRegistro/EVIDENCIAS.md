# Evidencias de la aplicación

Para completar la entrega, ejecutar la aplicación y crear un documento PDF con estas evidencias:

1. Pantalla inicial de la aplicación.
2. Selección de fecha del juego.
3. Selección del partido.
4. Selección de asiento disponible.
5. Formulario con los datos personales llenos.
6. Comprobante generado.
7. Ventana de impresión usando la opción Guardar como PDF.

El comprobante se genera desde la misma aplicación y puede guardarse como PDF usando el navegador.
